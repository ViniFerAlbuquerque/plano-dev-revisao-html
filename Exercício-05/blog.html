Introdução ao HTML5

Publicado em 17/11/2025

HTML5 trouxe muitas novidades para o desenvolvimento web...

`**HTML significa:**
→ HyperText Markup Language (Linguagem de Marcação de Hypertexto).
- Linguagem: Forma de comunicação. No nosso caso, entre o programador e o navegador de Internet (Google Chrome, Firefox, Safari, Edge, Internet Explorer…)
- Marcação: O HTML ‘marca’ o que é o que em uma página. Ele que vai dizer o que é um texto, o que é uma imagem, um vídeo e assim por diante.
- Hypertexto: Hypertexto faz referência a escritá gráfica, sintetizando, é um nome bonito para site.

Resumindo…HTML é a tecnologia que fala o que é cada item em uma página de internet / um site. E ele fez isso por meio de TAG's.
TAG's ? Tag's são etiquetas. São os trechos de código que nos ajudam a falar o que é o que em uma página.`

JavaScript ES6

JavaScript é uma linguagem de programação amplamente utilizada no desenvolvimento web.
Ela permite adicionar interatividade e dinamismo às páginas da web. A sigla DOM significa "Document Object Model" (Modelo de Objetos do Documento) e refere-se à representação da estrutura de um documento HTML em forma de árvore. O JavaScript interage com a DOM para manipular os elementos da página, como alterar o conteúdo, estilos, adicionar ou remover elementos, manipular eventos, entre outros.
